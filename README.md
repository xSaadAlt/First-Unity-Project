This project marks the beginning of my journey into game development using the Unity engine. The goal is to master the fundamentals and understand how the engine works through hands-on practice.

## 🛠️ Tools & Technologies
- **Engine:** Unity
- **Language:** C#
- **Key Tools:** Cinemachine for camera control, UI System, Audio Source.

## 🎯 Learning Objectives
- [x] Implemented the Start/Main Menu UI.
- [x] Set up background music (BGM).
- [x] Develop basic AI for enemies (Movement & Animations).
- [ ] Master player movement (Movement & Double Jump).
- [ ] Understand the Animation system (Animator Controller) for the player.
- [ ] Set up professional camera systems (Cinemachine).
